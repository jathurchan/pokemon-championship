◆ operator>>()

JSONCPP_ISTREAM & Json::operator>> ( JSONCPP_ISTREAM &  sin,
Value &  root 
)

Read from 'sin' into 'root'.

Always keep comments from the input JSON.

This can be used to read a file into a particular sub-object. For example:

Json::Value root;
cin >> root["dir"]["file"];
cout << root;
